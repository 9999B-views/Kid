Neden Code Notes?

“Code Notes” çocuklarla programlama eğitimini modüler bloklar üzerinden, az bütçeye sahip ortamlarda çalışabileceğimiz bir ortam sunan kodlama platformudur. Bu platform çocuklara basit programlama bloklarını öğretebilme fonksiyonelliğe sahip bir programlama dili ile birlikte, bu dilin çıktılarını derleyebileceğimiz mobil uygulamadan oluşmaktadır. Code Notes, Türkçe karşılığı ile “Kodlama Kartları”, adını programlama dilini oluşturan ucuz maliyetli kartlardan alır.

Programlama Blokları

Start-End Blokları
Code Notes programları START kartı ile başlar, END kartı ile biter. Yazdığınız program bu iki kartın arasında değilse uygulama kodunuzu algılayamaz.
IF Bloğu
IF kartının yanına koyacağınız kart bir mantık önermesi olmalıdır. Bu mantık önermesinin doğruluk veya yanlışlığına göre altındaki kart programa çıktı olur.
Mantık İfadeleri
Örneğin yukarıdaki kart doğru bir ifadeydi, bunun sonucunda telefonda animasyonlu bir küp gördünüz. Fakat yandaki ifadedeki mantık önermesi yanlış olduğu için program bir çıktı vermez.
ELSE Kartı
Yanlış bir mantık önermesi kullandığımız durumlarda program çıktısı almak istiyorsak ELSE kartını kullanmalıyız. ELSE kartı Code Notes sıralı dizisinde tek başına durmalıdır.
FOR Döngüsü
Programlamada bazen aynı şeyleri birçok kez yaptırmak isteriz. Code Notes’ta böyle ifadeleri FOR kartının yanına koyacağımız kaç kere olacağını belirten «x times» kartı ile sağlarız.
WHILE Döngüsü
Code Notes’ta While kartının yanına gelebilecek tek önerme «end of screen»dir. Bu ifade ekranın sonuna kadar yazdırmak istediğiniz şeyi yazdırır.

Teori ve Tasarım

Programlama eğitiminin müfredatta kendine yer edinmesi ve öneminin giderek daha iyi anlaşılması ile beraber, programlama öğreten platformların öğrenme teorilerinden yararlanma ihtiyacı giderek artmıştır. Eğitim ortamlarında uygulanan aktiviteler, çocukların sınıf ortamında birbirleriyle etkileşerek ve kendi kendilerine bilgisayarca düşünme becerileri kazanmasında büyük öneme sahiptir. İki öğrenme tipinde de öğrencinin sınıf arkadaşları, öğretmeni ve eğitim materyali ile etkileşimi çok etkilidir. Code Notes açık-kaynak bir programlama dili ve Android uygulamsı ile beraber sunduğumuz, ucuz ve modüler bir platformdur. Code Notes hem fiziksel olarak temas etme imkanı verir hem de kullanıcıya fiyat olarak erişilebilir bir çözüm sunar.
